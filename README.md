# EDS Power API

## Призначення

EDS Power — модульна система для управління виробничими процесами.

Поточний активний сервер:
- Render API: https://eds-power-api.onrender.com

## Головне правило

Репозиторій є єдиним джерелом правди для:
- коду;
- архітектури;
- описів модулів;
- правил для AI;
- технічних рішень.

## Поточні модулі

1. 00-01_AUTH — авторизація
2. 00-02_CALC_CONFIGURATOR — конфігуратор розрахунків
