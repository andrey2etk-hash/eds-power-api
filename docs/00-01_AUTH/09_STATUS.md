# Status

## Поточний статус

draft_ready

AUTH frozen at `draft_ready`.

Це означає:

- структура модуля зафіксована
- нові AUTH-функції не додаються на Stage 1
- дозволені тільки виправлення Markdown, контрактів або явних конфліктів
- подальше розширення AUTH можливе тільки через TASK

## Що входить у перший прототип

- базова таблиця users
- базові ролі
- перевірка користувача
- передача user_id в API-запити

## Що не входить у перший прототип

- складна адмін-панель
- детальні права на кожну кнопку
- двофакторна авторизація
