# 00-01_AUTH Module Overview

## Призначення

Модуль `00-01_AUTH` відповідає за авторизацію користувачів, ролі, права доступу та ідентифікацію користувача в системі EDS Power.

## Основна задача

Забезпечити, щоб кожна дія в системі виконувалась від імені конкретного користувача з визначеною роллю та дозволами.

## Що модуль робить

- зберігає користувачів
- визначає роль користувача
- перевіряє статус користувача
- передає `user_id` в інші модулі
- перевіряє права доступу

## Що модуль не робить

- не виконує розрахунки
- не формує КП
- не керує виробництвом
- не змінює дані інших модулів напряму
