# Module Links

## AUTH

`00-02_CALC_CONFIGURATOR` використовує `00-01_AUTH` для:

- визначення `user_id`
- перевірки активної сесії
- перевірки прав на дію

## COMMERCIAL_OFFERS

У майбутньому результат розрахунку може використовуватись модулем `00-03_COMMERCIAL_OFFERS`.

На Stage 2 preparation ця інтеграція не реалізується.

## PRODUCTION_TRANSFER

У майбутньому результат розрахунку може бути основою для передачі у виробництво.

На Stage 2 preparation ця інтеграція не реалізується.

## Правила взаємодії

- модуль працює тільки через API
- модуль не змінює дані інших модулів напряму
- інтеграції додаються тільки після TASK
- обмін даними відбувається тільки через data contracts
