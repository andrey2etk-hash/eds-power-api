# UI UX

## Stage 2 preparation

UI для `00-02_CALC_CONFIGURATOR` ще не реалізується.

Цей файл фіксує тільки базові очікування для майбутнього UI / GAS шару.

## Майбутні екрани

- форма створення розрахунку
- форма введення параметрів
- перегляд статусу розрахунку
- перегляд результату або помилки

## Базові поля

- object number
- product type
- common parameters
- product specific parameters
- options

## Validation responsibility

UI / GAS може:

- перевіряти заповнення обов’язкових полів
- показувати локальні повідомлення
- не відправляти порожній запит

UI / GAS не може:

- виконувати розрахунки
- вибирати алгоритм
- викликати validation або normalization як окремі API дії
- змінювати структуру `payload`
- приймати фінальне рішення про валідність даних

Фінальна перевірка виконується API.
