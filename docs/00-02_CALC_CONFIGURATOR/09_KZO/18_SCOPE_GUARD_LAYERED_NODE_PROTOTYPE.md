# KZO Layered Node Prototype Scope Guard

Status: `ACTIVE_GUARD`

## Allowed (planning/prototype framing only)

- bounded family hierarchy definition
- bounded role/position influence mapping
- one prototype family
- one demo case
- layered-node doctrine framing

## Forbidden

- implementation code
- broad KZO expansion
- full BOM expansion
- API redesign
- GAS logic changes
- DB redesign
- pricing / ERP / production transfer logic
