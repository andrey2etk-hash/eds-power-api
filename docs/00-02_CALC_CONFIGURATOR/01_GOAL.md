# Goal

## Ціль модуля

Підготувати базову структуру для розрахунків виробів у системі EDS Power.

Модуль повинен відповідати на питання:

- який об’єкт розраховується
- який тип виробу обрано
- які параметри передані для розрахунку
- чи пройшли параметри перевірку
- який статус має розрахунок
- який результат повертається користувачу

## Межі Stage 2 preparation

На цьому етапі модуль описує тільки generic CALC structure.

Продуктова логіка, конкретні алгоритми та детальні формули переносяться в окремі TASK перед MVP.
