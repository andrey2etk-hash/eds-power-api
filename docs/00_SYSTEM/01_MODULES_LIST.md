# EDS Power Modules List

## 1. Основні модулі системи

### `00-01_AUTH`
Авторизація та управління користувачами.

Включає:

- реєстрацію користувачів
- вхід у систему
- ролі
- права доступу

---

### `00-02_CALC_CONFIGURATOR`
Конфігуратор розрахунків виробів.

Включає:

- вибір типу виробу
- введення параметрів
- запуск розрахунку
- формування результату

---

### `00-03_COMMERCIAL_OFFERS`
Формування комерційних пропозицій.

Включає:

- генерацію комерційних пропозицій
- підстановку параметрів
- роботу з шаблонами
- формування документів

---

### `00-04_PRODUCTION_TRANSFER`
Передача результатів у виробництво.

Включає:

- створення виробничих задач
- підготовку даних для виробництва
- передачу в систему виробництва

---

### `00-05_KITTING`
Комплектація.

Включає:

- підготовку комплектуючих
- формування наборів
- перевірку наявності

---

### `00-06_SUPPLY`
Постачання.

Включає:

- закупівлі
- взаємодію з постачальниками
- контроль поставок

---

### `00-07_PRODUCTION`
Виробництво.

Включає:

- виконання виробничих задач
- контроль процесів
- відстеження статусів

---

### `00-08_ANALYTICS`
Аналітика системи.

Включає:

- аналіз часу
- аналіз витрат
- аналіз ефективності виробництва

---

## 2. Правила модулів

- кожен модуль має власну папку в `docs/`
- кожен модуль має описані:
  - входи (inputs)
  - виходи (outputs)
  - процес (flow)
- кожен модуль працює тільки через API
- модулі не дублюють логіку
- модулі взаємодіють тільки через data contracts
- один модуль не має прямого доступу до іншого без API
