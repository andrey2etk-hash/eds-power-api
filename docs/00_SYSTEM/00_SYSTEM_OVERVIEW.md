# EDS Power System Overview

## 1. Загальна мета системи

EDS Power — це модульна система для управління виробничими процесами підприємства.

Система об'єднує:
- розрахунки виробів
- формування комерційних пропозицій
- передачу у виробництво
- комплектацію
- постачання
- виробництво
- аналітику

---

## 2. Принцип побудови

Система будується як набір незалежних модулів.

Кожен модуль:
- має власну зону відповідальності
- має описані входи та виходи
- працює через API
- не порушує інші модулі

---

## 3. Основні модулі системи

1. 00-01_AUTH — авторизація та користувачі  
2. 00-02_CALC_CONFIGURATOR — конфігуратор розрахунків  
3. COMMERCIAL_OFFERS — комерційні пропозиції  
4. PRODUCTION_TRANSFER — передача у виробництво  
5. KITTING — комплектація  
6. SUPPLY — постачання  
7. PRODUCTION — виробництво  
8. ANALYTICS — аналітика  

---

## 4. Архітектура

Система побудована за принципом:

UI (Google Sheets / Web)
↓
GAS (тонкий клієнт)
↓
API (Render)
↓
Database (Supabase)

---

## 5. Принцип роботи

Будь-яка дія в системі:

1. Користувач вводить дані  
2. Дані проходять перевірку  
3. Дані нормалізуються  
4. Обирається алгоритм  
5. Виконується розрахунок  
6. Результат зберігається  
7. Повертається статус  

---

## 6. Правило системи

Репозиторій є єдиним джерелом правди.

Будь-яка зміна:
- спочатку описується
- потім реалізується
